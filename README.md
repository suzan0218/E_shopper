# E-Shopper Web Application

## Overview
E-Shopper is a web-based application that simulates an online shopping system.  
It allows users to browse products, manage a cart, and perform basic shopping operations.

---

## Features
- User login and registration  
- Product listing  
- Shopping cart functionality  
- Basic order handling  

---

## Tech Stack
- Java  
- MySQL  
- HTML, CSS, JavaScript  

---

## Description
This project focuses on backend logic and database integration using Java and MySQL.  
It demonstrates my understanding of building and managing a simple web application.
